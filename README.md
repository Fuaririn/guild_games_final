# GuildGameStructured (Avalonia)

## Lancer
Dans ce dossier :

```bash
dotnet restore
dotnet run
```

## Arborescence
- `Models/` : données (Hero, Mission, GuildState…)
- `Services/` : moteur de jeu (GameEngine) + événements aléatoires
- `ViewModels/` : logique UI (MVVM)
- `Views/` : AXAML + code-behind
